<h2>🛠️ passos de instalação:</h2>

<p>1. Clonar repositório</p>

```
git clone https://github.com/Somma11/Ticket-trident
```

<p>2. Instalar dependências</p>

```
npm i
```

<p>3. configurar o arquivo config.json</p>

<p>4. iniciar</p>

```
node .
```
